Thư mục kết quả thực nghiệm đã tạo từ dataset data1.csv, data2.csv và data3.csv.
Các file chính:
- summary.csv: thống kê tổng quan.
- top_keywords.csv: top item/khía cạnh.
- frequent_itemsets.csv: tập item phổ biến.
- association_rules.csv: toàn bộ luật kết hợp.
- sentiment_rules.csv: luật có vế phải là sentiment.

 Các thư viện cần cài đặt:
pandas, matplotlib, itertools, collections
